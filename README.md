# SimpleVideoEdit
- a simple android video edit app and sdk
- Android平台视频编辑APP以及相应的SDK


## Feature
- OpenGL Filters
- Hardware Transcoding
- Video Timeline Edit
- OpenGL滤镜
- 硬件转码
- 视频时间线编辑

## Plan
- Audio Replace
- 音乐替换
